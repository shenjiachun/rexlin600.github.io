## 4 文件配置（换端口）

有些朋友安装后可能出现 502 错误，这种一般是端口被占用了，可以改成下面这种

### 4.1 修改 gitlab-http.conf 文件

进入这个目录 `cd /var/opt/gitlab/nginx/conf`

修改端口：

```
listen*:80

修改为：
listen*:82
```

### 4.2 修改 gitlab.rb

进入这个目录`/etc/gitlab`

```
nicorn['port'] = 80

修改为：
nicorn['port'] = 8088
```

```
nginx['listen_port'] = nil

修改为：
nginx['listen_port'] = 82
```

最后，再重新配置 gitlab、重启即可，如下：

```
gitlab-ctl reconfigure

gitlab restart
```
